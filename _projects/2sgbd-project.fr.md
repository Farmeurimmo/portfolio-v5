## SupCount Database Redesign

Projet individuel réalisé dans le cadre du module de bases de données de SUPINFO.

L'objectif était de repenser entièrement l'architecture de données d'une application de partage de dépenses afin de
proposer une solution cohérente, évolutive et adaptée à un contexte professionnel.

Note : **100.00**

### Contexte et exigences

Le projet consistait à analyser les besoins fonctionnels d'une application fintech puis à concevoir une base de données
complète répondant aux problématiques de performance, de maintenance et d'évolution.

L'ensemble du travail a été réalisé individuellement sur une durée de deux semaines.

### Réalisations principales

* Conception du dictionnaire de données.
* Création des modèles MCD, MLD et MPD.
* Implémentation complète de la base de données sous MySQL.
* Création de jeux de données de démonstration.
* Mise en place de vues SQL dédiées à l'analyse métier.
* Développement de procédures stockées.
* Création de triggers et d'événements planifiés.
* Gestion des rôles, utilisateurs et permissions.
* Optimisation via indexation.
* Étude d'une architecture de réplication source-réplica.

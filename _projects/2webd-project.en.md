## RAWG Explorer

Web application designed to discover and search for video games using the RAWG API.

Grade: **82.20**

### Context and requirements

This project was developed in a team of two as part of a web development course.

The objective was to create an application capable of exploring a large video game catalog while applying frontend development best practices.

### Main features

* Video game search.
* Result filtering and sorting.
* Detailed game information pages.
* Navigation between publishers and their games.
* Persistent favorites management.
* Custom error pages.

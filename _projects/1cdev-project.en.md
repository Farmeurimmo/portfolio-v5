### Context and Requirements

Develop a combat game in C with entities called supémons, each having stats, attacks and status effects.

### Deliverables

- Structures for supémons and attacks.
- Functions to handle combats, attacks and status effects.
- Console interface to display combat state and options.
- User input handling with validation.
- Turn-based system alternating between player and AI.
- Victory/defeat conditions based on supémon HP.
- Possible capture of new supémons after combat.
- Evolution of supémons after winning battles.
- Healing center to restore supémon HP.

### Grading Criteria

- Correctness of structures and functions.
- Smooth combat flow with turn management and effects.
- Clear user interface.
- Robust validation of user inputs.
- Respect of technical constraints (C language).

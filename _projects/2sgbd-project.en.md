## SupCount Database Redesign

Individual project completed as part of the SUPINFO database module.

The objective was to completely redesign the data architecture of an expense-sharing application and provide a scalable,
maintainable, and production-ready database solution.

Grade: **100.00**

### Context and Requirements

The project focused on analyzing business requirements and designing a complete database architecture capable of
supporting future growth, performance improvements, and long-term maintainability.

The entire project was completed individually over a two-week period.

### Main Achievements

* Data dictionary design.
* Conceptual, logical, and physical data modeling.
* Full MySQL database implementation.
* Sample dataset generation.
* Business-oriented SQL views.
* Stored procedure development.
* Trigger and scheduled event implementation.
* User roles and permissions management.
* Database optimization through indexing.
* Source-replica replication architecture study.

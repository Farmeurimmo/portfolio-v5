## Mini-projet cinéma

Application web pour afficher et rechercher des films à l'aide de l'API OMDb.

### Contexte et exigences

Un cinéma de village souhaite une application pour permettre aux clients de voir les films disponibles via une API.

### Livrables

- Site web avec au moins 3 pages : index, recherche, film.
- Utilisation d'une clé API pour accéder à OMDb.
- Afficher les films tendance sur la page d'accueil.
- Recherche en temps réel sans rechargement de page.
- Afficher les informations détaillées du film.
- Utiliser un serveur HTTP local (Live Server ou serve).
- Code organisé avec plusieurs fichiers JS/TS et un bundler.
- Bundle séparé par page recommandé.

### Critères de notation

- index : afficher les films tendance avec des informations et un bouton 'charger plus'.
- recherche : barre de recherche et résultats en temps réel, bouton 'charger plus'.
- film : détails complets du film, synopsis, genre, acteurs, notes et date de sortie DVD en bonus.
- Gestion Git : dépôt distant, branches et messages de commit.

## Sports Tournament Management Platform

Web application designed to organize and manage sports tournaments through a REST API.

Grade: **91.75**

### Context and requirements

This project was developed as a team project for an advanced PHP course.

The objective was to build a complete tournament management platform while following REST principles, security best
practices, and Symfony architecture standards.

### Main features

* Sports tournament management.
* Player and user account management.
* Tournament registration management.
* Match and score management.
* Secure JWT authentication.
* Administration interface.
* Automated notification system.
* Custom Symfony commands.

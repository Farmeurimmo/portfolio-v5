## Cinema Mini-Project

Web application to display and search movies using the OMDb API.

### Context and Requirements

A village cinema wants an app to allow clients to view available movies via an API.

### Deliverables

- Website with at least 3 pages: index, search, movie.
- Use of an API key to access OMDb.
- Display trending movies on homepage.
- Real-time search with no page reload.
- Display detailed movie information.
- Use a local HTTP server (Live Server or serve).
- Organized code with multiple JS/TS files and bundler.
- Separate bundle per page recommended.

### Grading Criteria

- index: display trending movies with info and 'load more' button.
- search: search bar and real-time results, 'load more' button.
- movie: full movie details, synopsis, genre, actors, ratings and DVD date as bonus.
- Git management: remote repo, branches and commit messages.
